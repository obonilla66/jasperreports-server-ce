Download Location for SQLServer JDBC Driver
-------------------------------------------

  http://www.microsoft.com/en-us/download/details.aspx?id=11774


Copy the downloaded driver jar to the following location:

  <js-install>/buildomatic/conf_source/db/sqlserver/jdbc


For additional information on using JDBC jar files please refer to the
JasperReports Server Installation Guide which is located here:

  <js-install>/docs/JasperReports-Server-Install-Guide.pdf

  See Chapter 5 "Installing the WAR File Distribution"
